Este projeto foi desenvolvido com o objetivo de entender e aplicar as quatro operações básicas de um sistema de gerenciamento de dados: **CRUD** (Create, Read, Update e Delete). A aplicação utiliza PHP para a lógica de back-end e MySQL para o armazenamento das informações.

## ⚙️ Funcionalidades do CRUD

O sistema permite realizar as seguintes ações no banco de dados:

* **Create (Cadastrar):** Inserção de novos registros através de formulários.
* **Read (Listar):** Exibição dos dados armazenados em uma tabela organizada.
* **Update (Editar):** Alteração de informações de registros já existentes.
* **Delete (Excluir):** Remoção de registros do banco de dados com segurança.

## 🚀 Tecnologias e Ferramentas

* **Linguagem:** JavaScript
* **Ambiente:** Node.js
* **Bibliotecas:** Express e JSON Server
* **Interface:** HTML

## 🔰Como inicializar o projeto

- Faça o download do arquivo pelo github e abra a pasta no terminal.

- Instalar o node pelo link: 

- Você pode verificar se o node ja está instalado com o comando no terminal; <br>
``` 
npm- -v 
```

- Executar o seguinte comando no terminal para instalar os node-modules:
``` 
npm install json-server express 
``` 

- Iniciar o programa com o comando:
``` 
node server.js
```
